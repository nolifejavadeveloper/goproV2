# Introduction

**Gopro** (go proxy) is a lightweight Minecraft reverse proxy written in GO. Gopro has:
- Built-in plugin system
- Built-in event/packet system
With gopro, you can:
- Thanks to the built-in packet event system, you can implement a packet translator to make it multi-version compatible or even cross edition!
- Customize it to make it suit your own needs
